# \<tiny-stepper>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i tiny-stepper
```

## Usage
```html
<script type="module">
  import 'tiny-stepper/tiny-stepper.js';
</script>

<tiny-stepper></tiny-stepper>
```
